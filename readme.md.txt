# Amazon Sales Data Analysis

This project analyzes Amazon sales data using Python to derive meaningful business insights. 
The analysis covers sales performance, revenue trends, top-performing products, and customer behavior.
 It’s designed as a practical data science project using Jupyter Notebooks.

# Project Goals

 Clean and preprocess raw sales data
 Explore and visualize key trends
 Identify top-selling products and categories
 Analyze monthly revenue patterns
 Generate insights to support business decisions

#Tools & Technologies Used

*Python 3.x
*Jupyter Notebook
*Pandas– for data manipulation
*NumPy– for numerical operations
*Matplotlib & Seaborn – for data visualization
